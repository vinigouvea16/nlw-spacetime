## 🖥️ Projeto
Esse é um projeto responsivo de uma cápsula do tempo para exibir memórias em uma timeline/linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnlogias:
- HTML
- CSS
- Git e Github
